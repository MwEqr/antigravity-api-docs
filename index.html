🔑 Autenticação

Obtenha uma API Key grátis em aistudio.google.com/apikey.

URL Base
https://generativelanguage.googleapis.com/v1beta/models/{MODEL}:generateContent?key=API_KEY
⚠️ Segurança: Nunca exponha sua API Key no frontend público.

📤 Gerar Conteúdo

POST /v1beta/models/{model}:generateContent
Campo Tipo Status Descrição
contents array obrigatório Array de mensagens (role + parts)
generationConfig object opcional temperature, maxOutputTokens
systemInstruction object opcional Instrução de sistema
cURL
curl -X POST \
  "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{"role":"user","parts":[{"text":"Olá, como vai?"}]}]
  }'

🖼️ Enviar Imagens

Envie Base64 no campo inline_data:

JSON Body
{"contents":[{"role":"user","parts":[
  {"text":"Descreva esta imagem"},
  {"inline_data":{"mime_type":"image/jpeg","data":"BASE64_AQUI"}}
]}]}

Formatos: image/jpeg, image/png, image/webp, image/gif, application/pdf

🌊 Streaming

POST /v1beta/models/{model}:streamGenerateContent?alt=sse

Receba tokens em tempo real via Server-Sent Events.

📋 Exemplos

🐍 Python

Python
import requests, base64
API_KEY = "SUA_KEY"
MODEL = "gemini-2.0-flash"

def ask(prompt, image_path=None):
    url = f"https://generativelanguage.googleapis.com/v1beta/models/{MODEL}:generateContent?key={API_KEY}"
    parts = [{"text": prompt}]
    if image_path:
        b64 = base64.b64encode(open(image_path,"rb").read()).decode()
        parts.append({"inline_data":{"mime_type":"image/jpeg","data":b64}})
    r = requests.post(url, json={"contents":[{"role":"user","parts":parts}]})
    return r.json()["candidates"][0]["content"]["parts"][0]["text"]

print(ask("Relatório do servidor"))

📦 JavaScript

JavaScript
const API_KEY = 'SUA_KEY';
const MODEL = 'gemini-2.0-flash';

async function ask(prompt) {
  const r = await fetch(
    `https://generativelanguage.googleapis.com/v1beta/models/${MODEL}:generateContent?key=${API_KEY}`,
    { method:'POST', headers:{'Content-Type':'application/json'},
      body: JSON.stringify({contents:[{role:'user',parts:[{text:prompt}]}]}) }
  );
  const d = await r.json();
  return d.candidates[0].content.parts[0].text;
}
ask('Olá!').then(console.log);

🐘 PHP

PHP
<?php
$key = 'SUA_KEY';
$model = 'gemini-2.0-flash';
$ch = curl_init("https://generativelanguage.googleapis.com/v1beta/models/$model:generateContent?key=$key");
curl_setopt_array($ch, [
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => ['Content-Type: application/json'],
    CURLOPT_POSTFIELDS => json_encode(['contents'=>[['role'=>'user','parts'=>[['text'=>'Olá!']]]]]),
    CURLOPT_RETURNTRANSFER => true
]);
$r = json_decode(curl_exec($ch), true);
echo $r['candidates'][0]['content']['parts'][0]['text'];
