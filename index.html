Antigravity API

Documentação completa para interagir com os modelos Gemini programaticamente. Envie prompts, anexe imagens, escolha entre modo rápido ou planejamento, e receba respostas inteligentes.

API Operacional REST API Multimodal Streaming

🔑 Autenticação

Todas as requisições exigem uma API Key do Google AI Studio. Obtenha a sua em aistudio.google.com/apikey (grátis).

A chave é passada como query parameter key na URL:

URL
https://generativelanguage.googleapis.com/v1beta/models/{MODEL}:generateContent?key=SUA_API_KEY
⚠️ Segurança: Nunca exponha sua API Key no frontend. Use-a apenas no backend/servidor.

📤 Gerar Conteúdo

POST /v1beta/models/{model}:generateContent

Parâmetros do Body (JSON)

Campo Tipo Status Descrição
contents array obrigatório Array de mensagens com role e parts
contents[].role string obrigatório "user" ou "model"
contents[].parts array obrigatório Array com text e/ou inline_data
generationConfig object opcional Controla temperature, maxOutputTokens, etc.
systemInstruction object opcional Instrução de sistema (persona/contexto)

Exemplo Básico – Texto Simples

cURL
curl -X POST \
  "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{
      "role": "user",
      "parts": [{"text": "Explique o que é reverse proxy em 3 linhas"}]
    }]
  }'

Resposta

JSON Response
{
  "candidates": [{
    "content": {

    },

  }],
  "usageMetadata": {
    "promptTokenCount": 12,
    "candidatesTokenCount": 85,
    "totalTokenCount": 97
  }
}

🖼️ Enviar Imagens

O Gemini é multimodal — envie imagens junto com texto usando inline_data em Base64:

cURL – Imagem + Texto
# Converta a imagem para Base64 primeiro:
# BASE64=$(base64 -w0 imagem.jpg)

curl -X POST \
  "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{
      "role": "user",
      "parts": [
        {"text": "Descreva o que você vê nesta imagem"},
        {
          "inline_data": {
            "mime_type": "image/jpeg",
            "data": "'$BASE64'"
          }
        }
      ]
    }]
  }'

MIME Types Suportados

Formato MIME Type
JPEG image/jpeg
PNG image/png
WebP image/webp
GIF image/gif
PDF application/pdf
💡 Múltiplas imagens: Adicione vários objetos inline_data no array parts para enviar mais de uma imagem.

🌊 Streaming (Resposta em Tempo Real)

Use streamGenerateContent para receber a resposta token por token via SSE:

POST /v1beta/models/{model}:streamGenerateContent?alt=sse
cURL – Streaming
curl -X POST \
  "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-pro:streamGenerateContent?alt=sse&key=SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{
      "role": "user",
      "parts": [{"text": "Crie um site landing page em HTML"}]
    }]
  }'

📋 Exemplos Completos

🐍 Python

Python
import requests, base64, json

API_KEY = "SUA_API_KEY"
MODEL = "gemini-2.0-flash"  # ou "gemini-2.5-pro" para planning

def ask(prompt, image_path=None):
    url = f"https://generativelanguage.googleapis.com/v1beta/models/{MODEL}:generateContent?key={API_KEY}"
    
    parts = [{"text": prompt}]
    
    if image_path:
        with open(image_path, "rb") as f:
            b64 = base64.b64encode(f.read()).decode()
        parts.append({
            "inline_data": {"mime_type": "image/jpeg", "data": b64}
        })
    
    body = {
        "contents": [{"role": "user", "parts": parts}],
        "generationConfig": {
            "temperature": 0.7,
            "maxOutputTokens": 8192
        }
    }
    
    r = requests.post(url, json=body)
    data = r.json()
    return data["candidates"][0]["content"]["parts"][0]["text"]

# Uso simples (Fast mode)
print(ask("O que é Docker?"))

# Com imagem
print(ask("Descreva essa imagem", "foto.jpg"))

📦 JavaScript (Node.js)

JavaScript
const fs = require('fs');

const API_KEY = 'SUA_API_KEY';
const MODEL = 'gemini-2.0-flash'; // Fast mode
// const MODEL = 'gemini-2.5-pro'; // Planning mode

async function ask(prompt, imagePath = null) {
  const parts = [{ text: prompt }];
  
  if (imagePath) {
    const b64 = fs.readFileSync(imagePath).toString('base64');
    parts.push({
      inline_data: { mime_type: 'image/jpeg', data: b64 }
    });
  }
  
  const res = await fetch(
    `https://generativelanguage.googleapis.com/v1beta/models/${MODEL}:generateContent?key=${API_KEY}`,
    {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        contents: [{ role: 'user', parts }],
        generationConfig: { temperature: 0.7, maxOutputTokens: 8192 }
      })
    }
  );
  
  const data = await res.json();
  return data.candidates[0].content.parts[0].text;
}

// Uso
ask('Crie um componente React de login').then(console.log);

🐘 PHP

PHP
<?php
$apiKey = 'SUA_API_KEY';
$model  = 'gemini-2.0-flash'; // Fast | 'gemini-2.5-pro' para Planning

function askGemini(string $prompt, ?string $imagePath = null): string {
    global $apiKey, $model;
    
    $parts = [['text' => $prompt]];
    
    if ($imagePath) {
        $b64 = base64_encode(file_get_contents($imagePath));
        $parts[] = [
            'inline_data' => [
                'mime_type' => 'image/jpeg',
                'data' => $b64
            ]
        ];
    }
    
    $body = json_encode([
        'contents' => [['role' => 'user', 'parts' => $parts]],
        'generationConfig' => [
            'temperature' => 0.7,
            'maxOutputTokens' => 8192
        ]
    ]);
    
    $ch = curl_init("https://generativelanguage.googleapis.com/v1beta/models/{$model}:generateContent?key={$apiKey}");
    curl_setopt_array($ch, [
        CURLOPT_POST => true,
        CURLOPT_HTTPHEADER => ['Content-Type: application/json'],
        CURLOPT_POSTFIELDS => $body,
        CURLOPT_RETURNTRANSFER => true,
        CURLOPT_TIMEOUT => 120
    ]);
    
    $response = curl_exec($ch);
    curl_close($ch);
    
    $data = json_decode($response, true);
    return $data['candidates'][0]['content']['parts'][0]['text'] ?? 'Erro na resposta';
}

// Uso
echo askGemini('Como configurar Nginx como reverse proxy?');
echo askGemini('O que tem nessa imagem?', '/caminho/foto.jpg');

❌ Códigos de Erro

Código Significado Solução
400 Request inválido Verifique o JSON do body
401 API Key inválida Gere uma nova key no AI Studio
403 Sem permissão Habilite a API no Google Cloud
429 Rate limit Aguarde e tente novamente (exponential backoff)
500 Erro do servidor Tente novamente em alguns segundos
