📖 Visão Geral

A API Antigravity é uma ponte direta para o agente IA interno rodando na VPS. Utiliza compatibilidade de payload similar ao formato OpenAI.

Online OpenAI-Compatible Multimodal

🔑 Autenticação

O endpoint de proxy fornecido (/api.php) autentica automaticamente de forma estática com o serviço interno. Não é necessária API Key para testar a comunicação local.

📤 Gerar Conteúdo (Chat Completions)

POST https://api-docs.encrypthost.com.br/api.php
Campo Tipo Status Descrição
model string obrigatório O ID do modelo, ex: "clawdbot" ou "gemini-3.1-pro"
messages array obrigatório Array de mensagens em formato OpenAI (role, content)
cURL
curl -X POST \
  "https://api-docs.encrypthost.com.br/api.php" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "clawdbot",
    "messages": [
      {"role":"system", "content":"Seja conciso."},
      {"role":"user", "content":"Olá, como vai?"}
    ]
  }'

🖼️ Enviar Imagens

JSON Array
{
  "model": "clawdbot",
  "messages": [{
    "role": "user",
    "content": [
      {"type":"text", "text":"Descreva esta imagem"},
      {"type":"image_url", "image_url":{"url":"data:image/jpeg;base64,AQUI"}}
    ]
  }]
}

🌊 Streaming

O endpoint via PHP por padrão retorna uma chamada em bloco único. O streaming de eventos WebSocket/SSE completo com o Gateway deve ser configurado via cliente Node WS direto na porta 18789 no futuro.

💬 Chat Multi-turno

Basta enviar o array de messages sucessivamente preenchido.

JSON
{
  "model":"clawdbot",
  "messages":[
    {"role":"user","content":"O que é Docker?"},
    {"role":"assistant","content":"Docker é uma plataforma..."},
    {"role":"user","content":"E Kubernetes?"}
  ]
}

📋 Exemplos

🐍 Python

Python
import requests

url = "https://api-docs.encrypthost.com.br/api.php"
data = {
    "model": "gemini-3.1-pro",
    "messages": [{"role": "user", "content": "Relatório do servidor"}]
}

r = requests.post(url, json=data)
print(r.json()["choices"][0]["message"]["content"])

📦 JavaScript

JavaScript
async function ask(prompt) {
  const r = await fetch('https://api-docs.encrypthost.com.br/api.php', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      model: 'clawdbot',
      messages: [{ role: 'user', content: prompt }]
    })
  });
  const data = await r.json();
  return data.choices[0].message.content;
}

ask('Olá!').then(console.log);

🐘 PHP

PHP
<?php
$ch = curl_init("https://api-docs.encrypthost.com.br/api.php");
$data = ['model' => 'clawdbot', 'messages' => [['role'=>'user', 'content'=>'Olá!']]];

curl_setopt_array($ch, [
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => ['Content-Type: application/json'],
    CURLOPT_POSTFIELDS => json_encode($data),
    CURLOPT_RETURNTRANSFER => true
]);

$response = json_decode(curl_exec($ch), true);
echo $response['choices'][0]['message']['content'];
